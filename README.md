# SHAR speedrunning setup scripts for Linux
**\*\*CURRENTLY WIP\*\***

This is a repository for setting up some stuff for speedrunning SHAR (The Simpsons: Hit & Run) for Linux **but will not set up the installation of SHAR specifically**. Non-speedrunners may also get some use out of these scripts.

While these scripts won't install SHAR, they can be used to set up Wine prefixes for SHAR speedrunning, creating .desktop files and creating shell scripts to launch LiveSplit/SHAR.

I plan to implement LiveSplit installation at some point (although this is pretty simple anyway).
